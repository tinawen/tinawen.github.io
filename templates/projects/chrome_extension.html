Dropbox Picture Viewer Chrome Extension

This is a chrome extension that does the OAuth dance to authenticate with Dropbox, uses the Dropbox API to pull all the pictures out of a user's account, and displays them in a window to make them easily accessible. Drag and drop into Gmail or other windows is supported. The user can also configure what size image dropping should be.
